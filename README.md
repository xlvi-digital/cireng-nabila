# cireng-nabila
Cireng Isi Nabila adalah sebuah aplikasi kasir (Point of Sale) berbasis web sederhana yang dirancang khusus untuk pelaku UMKM kuliner. Aplikasi ini memfokuskan pada kemudahan pencatatan pesanan, manajemen riwayat penjualan, dan pelaporan pendapatan harian secara cepat dan efisien langsung dari perangkat mobile maupun desktop.
